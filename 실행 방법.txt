# Room_NPS 실행 가이드

## 1. 라이브러리 설치 (최초 1회)
* **백엔드 패키지 설치:**
  ```bash
  .\venv\Scripts\pip install -r requirements.txt
  ```
* **프론트엔드 패키지 설치:**
  ```bash
  cd frontend
  npm.cmd install
  cd ..
  ```

## 2. 백엔드 실행 방법
* 프로젝트 루트 경로에서 아래 명령어를 실행합니다.
  ```bash
  .\venv\Scripts\python app.py
  ```

## 3. 프론트엔드 실행 방법
* `frontend` 폴더에서 아래 명령어를 실행합니다.
  ```bash
  cd frontend
  npm.cmd run dev -- --port 5174
  ```
